Introduccion

Python es un lenguaje de programación de alto nivel que se destaca por su legibilidad y su versatilidad. Aquí tienes una breve introducción:

  • Características del lenguaje Python: Multiparadigma: Admite programación imperativa, orientada a objetos y funcional. Multiplataforma: Disponible en Windows, Linux y Mac OS. Dinámicamente tipado: El tipo de las variables se decide en tiempo de ejecución. Fuertemente tipado: No se puede usar una variable fuera de su tipo sin conversión. Interpretado: No se compila a lenguaje máquina; se ejecuta mediante un intérprete.
  • El intérprete de Python: Cuando instalas Python, se añade el comando python (o python3 para la versión 3.x) al path y se instala el intérprete correspondiente. Puedes verificar la versión de Python instalada ejecutando python3 en la consola. El resultado mostrará la versión del intérprete. En el intérprete, puedes escribir expresiones e instrucciones. Por ejemplo, 2 + 3 devuelve 5
  • Primer programa en Python: Los programas se escriben en archivos con extensión .py.

    Crea un archivo llamado suma.py con contenido como este:


    # suma.py resultado = 2 + 3 print("La suma es:", resultado)
Variables

    En Python, las variables son fundamentales para almacenar y manipular datos. Permíteme explicarte:

  • ¿Qué es una variable?

    Una variable es un espacio de memoria reservado para guardar un valor. Cada variable tiene un nombre único que nos permite acceder y modificar su contenido. Imagina una variable como un contenedor en la memoria de la computadora donde almacenamos datos

  • Asignación de valores a variables: Para asignar un valor a una variable, utilizamos el operador de asignación =. Por ejemplo:
    • # Asignamos el valor 42 a la variable 'edad' edad = 42
  • Tipos de datos: Cada valor pertenece a un tipo de dato específico (entero, cadena, flotante, etc.). Al asignar un valor a una variable, definimos su tipo de dato

    • nombre = "Juan" # 'nombre' es una cadena pi = 3.1416 # 'pi' es un flotante
  • Acceso y modificación: Podemos acceder al valor de una variable y modificarlo durante la ejecución del programa. Por ejemplo:

    • resultado = 2 + 3 print("La suma es:", resultado) # Muestra "La suma es: 5"
  • Buenas prácticas:

    Define las variables al principio del programa. Si intentas usar una variable no definida, obtendrás un error. En resumen, las variables en Python nos permiten manejar datos de manera eficiente y estructurada. ¡Explora más y diviértete programando!

Tipos de datos

En Python, existen varios tipos de datos que nos permiten manejar información de manera eficiente. Aquí tienes una breve descripción de algunos de los tipos más comunes:

  1. Enteros (int): Representan números enteros, como 1, 2, -3, etc. Ejemplo:
    2. edad = 25
  2. Flotantes (float): Representan números con decimales, como 3.14, -0.5, etc. Ejemplo:
    3. pi = 3.1416
  3. Cadenas (str): Son secuencias de caracteres, como “hola”, “mundo”, “Python”, etc. Ejemplo:
    4. nombre = "Juan"
  4. Booleanos (bool): Tienen dos valores posibles: True o False. Se utilizan para expresar condiciones lógicas. Ejemplo:
    5. es_mayor_de_edad = True
  5. Listas (list): Almacenan múltiples valores en orden. Pueden contener elementos de diferentes tipos. Ejemplo:
    6. numeros = [1, 2, 3, 4]
  6. Tuplas (tuple): Son similares a las listas, pero son inmutables (no se pueden modificar). Ejemplo:
    7. coordenadas = (10, 20)
  7. Diccionarios (dict): Asocian claves (keys) con valores (values). Útiles para representar datos estructurados. Ejemplo:
    8. persona = {"nombre": "Ana", "edad": 30}
  8. Conjuntos (set): Contienen elementos únicos y no tienen un orden específico. Útiles para operaciones matemáticas como intersección y unión. Ejemplo:
    9. colores = {"rojo", "verde", "azul"}
Operadores

Los operadores en Python son símbolos que indican al intérprete que realice operaciones específicas, como aritmética, comparación, lógica, entre otras. Aquí tienes una breve descripción de los diferentes tipos de operadores:

  • Operadores Aritméticos: Realizan cálculos matemáticos entre operandos. Ejemplos:
    Suma: 12 + 3 = 15
    Resta: 12 - 3 = 9
    Multiplicación: 12 * 3 = 36
    División: 12 / 3 = 4
    Módulo (resto de la división): 16 % 3 = 1
    Potencia: 12 ** 3 = 1728
    División entera: 18 // 5 = 3
  • Operadores Relacionales: Comparan valores y devuelven un resultado booleano (verdadero o falso). Ejemplos:
    • Mayor que (>): 12 > 3 devuelve verdadero.
    Menor que (<): 12 < 3 devuelve falso.
    Igual a (==): 12 == 3 devuelve falso.
    Mayor o igual que (>=): 12 >= 3 devuelve verdadero.
    Menor o igual que (<=): 12 <= 3 devuelve falso.
    No igual a (!=): 12 != 3 devuelve verdadero.
  • Operadores Bit a Bit: Realizan operaciones a nivel de bits. Ejemplos:
    • AND (&): 10 & 11 = 10 (en binario).
    OR (|): 10 | 11 = 11 (en binario).
    XOR (^): 10 ^ 11 = 01 (en binario).
    NOT (~): ~2 = -3 (en binario)
    Desplazamiento a la derecha (>>): 2 >> 3 = 0 (en binario)
    Desplazamiento a la izquierda (<<): 2 << 3 = 16 (en binario).
  • Operadores Lógicos, de Asignación, de Pertenencia y de Identidad también son parte de Python. Explora más y diviértete programando con estos operadores Python
Estructuras de control

En Python, las estructuras de control son fundamentales para dirigir el flujo de ejecución de un programa. Estas estructuras nos permiten tomar decisiones, repetir bloques de código y crear lógica más compleja. Aquí tienes una breve descripción de las principales estructuras de control:

Condicional if: Permite ejecutar un bloque de código si se cumple una condición.

temperatura = 25 if temperatura > 30: print("Hace calor") else: print("No hace tanto calor")

Estas estructuras son esenciales para escribir programas más complejos y funcionales. ¡Explora y experimenta con ellas aqui!

Funciones

En Python, una función es un bloque de código reutilizable que se puede ejecutar muchas veces dentro de tu programa. Las funciones te permiten crear soluciones más modulares y DRY (Don’t Repeat Yourself) para problemas complejos. Aquí tienes una breve descripción de cómo funcionan las funciones en Python:

Definición de una función: Una función se define mediante la palabra clave def, seguida de un nombre de función y paréntesis que pueden contener parámetros.

Por ejemplo:

def saludar(nombre): print("¡Hola,", nombre, "!")

Hay muchos otros tipos de funciones. En resumen, las funciones en Python te permiten escribir código más eficiente, legible y modular. ¡Explora y crea tus propias funciones aqui!
